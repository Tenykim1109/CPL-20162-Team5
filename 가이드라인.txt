*운영자/매장관리자 - 동일앱, 계정으로 권한 분리
*소비자 - 동일앱, 계정 필요 없음

<운영자 - 운영자 계정으로 로그인>
1. 신규 매장 정보 생성 기능
  - 생성 기능

2. 매장관리
  -기존 매장정보 수정 및 삭제 기능

3. 라벨관리
  -기존 매장에 등록된 라벨 디자인 추가, 수정 및 삭제 기능

4. 기존 매장 검색기능


<매장관리자 - 운영자에게 승인받은 계정으로 로그인>
1. 주문내역 확인 기능
  - 소비자 별 라벨 주문 목록 (최대 7일까지만 보여지고, 재인쇄 가능)

1-1. 라벨 주문 인쇄기능
1-2. 라벨 주문 삭제기능

2. 지나간 주문 목록 보기 기능 ( 7일이 지난 주문 목록들, 재인쇄 불가능 )  

3. 판매현황 확인 기능 (그래프나 차트 이용이 좋을 듯함)
  - 제품별(메뉴) 판매 현황 : 전체 판매 수량 / 나이대 판매 현황 / 성별 판매 현황 
  - 매출 추이 : 일별, 주별, 월별


<소비자 - 계정없이 어플 사용 가능>
1. 라벨 메이커 기능
  - 기본으로 제공된 디자인에 텍스트, 사진 넣는 기능

2. 간단한 개인정보 입력 기능
  - 이름, 연락처, 이메일 [여기까지 기존정보], 성별, 나이 => 이 후 판매현황 정리용도

3. 회원가입 기능
  - 소비자 전용 계정을 이용하여 좋아요, 댓글, 이전에 생성한 라벨 디자인 저장 기능
